## 2024-04-11 - Auto-updating dropdowns and Return key bindings
**Learning:** Users expect modern UI interactions where selections in dropdowns auto-update the content without needing an extra click, and form inputs can be submitted with the Return key. Tkinter apps benefit greatly from these event bindings (`<<ComboboxSelected>>` and `<Return>`).
**Action:** Apply event bindings to Comboboxes and Entry fields in future Tkinter interfaces to improve keyboard accessibility and smooth interactions.
