## 2024-04-14 - Auto-updating and Return Key UX Patterns
**Learning:** Adding auto-update functionality on drop-down selections (`<<ComboboxSelected>>`) and enabling Enter-key submission (`<Return>`) for text fields aligns Tkinter apps closer to modern web-app UX expectations. It saves users unnecessary clicks and makes navigation smoother.
**Action:** When designing forms and lookup tools, always bind selection changes to update functions and bind the Return key to the primary submit action to improve keyboard accessibility and general UX.
